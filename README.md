# cpixel

Minimal pixel-art editor scaffold inspired by cpaint.

Features
- 64x64 pixel canvas
- Left-drag to draw, right-drag to erase
- Color picker, pixel size (zoom) slider
- Clear and Export PNG

Dev
- Vite + React + TypeScript + Tailwind CSS

Run
1. Install deps
   pnpm install
2. Dev server
   pnpm dev
3. Build
   pnpm build
